# BCM Performance Tracker

A web-based performance and progress tracker for Biology, Chemistry, and Mathematics (BCM) students.  
This system allows an administrator to:

- Add and delete students
- Submit scores per subject and assessment
- Auto-generate KPIs (class average, best student, most improved)
- Visualize performance using charts
- Generate class rankings
- Export CSV data
- Store all data securely using Firebase Firestore
